# Cookie Manager

一个基于 WXT 和 React 的浏览器扩展，用于管理浏览器 Cookie。

## 功能特点

- 查看和管理浏览器 Cookie
- 支持按域名筛选 Cookie
- 支持添加、修改和删除 Cookie
- 支持导入/导出 Cookie 数据

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 打包
npm run zip
```

## 支持的浏览器

- Chrome
- Firefox

## 技术栈

- WXT (Web Extension Tools)
- React
- TypeScript
- Biome (代码格式化工具)
